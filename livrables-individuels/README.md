# Livrables individuels

Chaque membre du groupe doit produire **les trois** livrables suivants (notés
individuellement, 40 % de la note finale) :

1. **Journal de bord individuel** — contributions datées (modèle : [journal-de-bord-modele.md](journal-de-bord-modele.md)).
   Exemple pré-rempli (ESIS ILD) : [journal-de-bord-charaf-ILD.md](journal-de-bord-charaf-ILD.md).
2. **Note réflexive (1-2 pages)** — rôle, choix, difficultés, apprentissages, lien métier
   (guide : [note-reflexive-modele.md](note-reflexive-modele.md)).
3. **Évaluation par les pairs** — une fiche par coéquipier, **confidentielle**
   (modèle : [evaluation-pairs-modele.md](evaluation-pairs-modele.md)).

> ⚠️ La **traçabilité** (commits nominatifs, journal de bord) et la **prise de parole filmée**
> déterminent la note individuelle. Un nom sur un livrable ne suffit pas : il faut une trace.
